# API Search - JSON-LD

This repository stores JSON-LD semantic vocabulary records for the [APIs.io](https://apis.io) search network.

## About

JSON-LD records provide linked data context and semantic vocabulary definitions for APIs indexed by the network. They are generated from api-evangelist repos and used to build the `_jsonld/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [schemas](https://github.com/api-search/schemas) - JSON Schema references
- [vocabularies](https://github.com/api-search/vocabularies) - API vocabularies
- [network](https://github.com/api-search/network) - The APIs.io network site
